🚀 手動上傳到 GitHub - 詳細步驟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ PowerShell 可能有問題，請手動操作

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式 A：使用 Git Bash（推薦）⭐

1. 在資料夾空白處按右鍵
2. 選擇「Git Bash Here」
3. 執行以下命令：

   git add index.html register.html profile.html transfer.html edit.html history.html admin.html
   git add google-apps-script.js
   git add assets/style.css
   git add README.md .gitignore
   git commit -m "更新核心功能"
   git push origin main

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式 B：使用 VS Code 內建 Git

1. 在 VS Code 中開啟資料夾
2. 點擊左側「Source Control」圖示
3. 勾選要上傳的文件：
   ☑ index.html
   ☑ register.html
   ☑ profile.html
   ☑ transfer.html
   ☑ edit.html
   ☑ history.html
   ☑ admin.html
   ☑ google-apps-script.js
   ☑ assets/style.css
   ☑ README.md
   ☑ .gitignore

4. 輸入提交訊息：「更新核心功能」
5. 點擊「Commit」
6. 點擊「Push」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式 C：使用 GitHub Desktop

1. 開啟 GitHub Desktop
2. 選擇倉庫：member-system
3. 勾選要提交的文件（同上）
4. 填寫提交訊息
5. 點擊「Commit to main」
6. 點擊「Push origin」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式 D：直接在 GitHub 網頁上傳

1. 前往 https://github.com/sky770825/member-system
2. 點擊「Add file」→「Upload files」
3. 拖曳以下文件：
   - 所有 .html 文件
   - google-apps-script.js
   - assets/style.css
4. 填寫提交訊息
5. 點擊「Commit changes」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 確認上傳成功：

1. 訪問 https://github.com/sky770825/member-system
2. 確認看到：
   ✅ 7 個 HTML 文件
   ✅ google-apps-script.js
   ✅ assets 資料夾
   ✅ README.md

3. 確認看不到：
   ❌ 任何 .md 說明文件（除了 README.md）
   ❌ tools 資料夾
   ❌ templates 資料夾

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
